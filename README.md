# BMI Calculator

A simple and interactive BMI (Body Mass Index) calculator that allows users to input their height and weight, calculates their BMI using a function, and displays their health status based on standard BMI ranges.

---

## 🚀 Features
- Takes user input for **height** and **weight**
- Calculates **BMI using a function**
- Displays **BMI value**
- Shows **health status** using conditional statements:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Clean and user-friendly interface

---

## 🛠️ Tech Stack
- **HTML** – Structure  
- **CSS** – Styling  
- **JavaScript** – Logic & DOM manipulation  
- **Python** – Function-based BMI calculation logic  
- **VS Code**, **Git**, **GitHub**


---

## ⚙️ How It Works
1. User enters height and weight
2. A function calculates the BMI value
3. Conditional statements determine health status
4. Result is displayed dynamically on the page
